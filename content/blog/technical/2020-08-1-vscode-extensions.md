How do I get a cursor on every line of selected snippet in vscode ?

```markdown
alt + shift + i
```

How to search a text globally across the project ?

```markdown
cmd/ctrl + shift + f
```

How to search for a function ?

```md
cmd/ctrl + shift + o
```

How to search across the files ?

```md
cmd/ctrl + t
```
